# Booth Detail Page Redesign Plan
## Making Every Booth Page a Destination Worth Visiting

**Created:** 2025-12-02
**Purpose:** Transform booth detail pages from sparse information displays into compelling, content-rich experiences that inspire visits and build community.

---

## Executive Summary

Only 200 analog photobooths remain in the world, maintained by a tight-knit community of 50 enthusiasts. Booth Beacon has an opportunity to become the definitive resource for this passionate community by creating booth pages that celebrate both the machines and the culture around them.

**Current State:**
- 1,487 booths with 61% data completeness
- Functional but sparse pages
- Limited visual appeal
- Minimal engagement features

**Vision:**
Transform each booth page into a mini-museum exhibit that:
- Educates about the machine's history and specifications
- Inspires visits with compelling narratives
- Builds community through user contributions
- Preserves photobooth culture for future generations

---

## Part 1: Understanding Photobooth Culture

### What Excites People About Vintage Photobooths

Based on research from the photobooth community, preservation efforts, and recent museum openings:

#### 1. **The Tangible Magic**
*"I love the tangible memories that are created, and that you only have four shots."* - 28-year-old enthusiast

- **Physical keepsakes** in a digital world
- The **glossy strip** fresh from the developer
- Only 4 shots - no do-overs, no filters
- Something rare, beautiful, and rooted in decades of tradition
- The satisfying **clunk of the shutter** and anticipation of watching images develop

#### 2. **The Analog Experience**
- **Chemical process** that digital can't replicate
- Unique **image quality** with characteristic grain and contrast
- Real film creates a type of image that digital photobooths simply cannot match
- The mechanical experience - dip-and-dunk chemistry, strobe flashes, physical development

#### 3. **Nostalgia & Intergenerational Appeal**
- Younger generations (grew up with smartphones) discovering how beautiful physical photos are
- Older generations remembering high school dates, family moments, cultural milestones
- The century-old technology making a resurgence in the selfie era
- Connection to photobooth moments throughout history (The Beatles, Andy Warhol, family archives)

#### 4. **Rarity & Preservation**
- **Only 200 working analog photobooths left worldwide**
- Each booth is a piece of living history
- Tightly-knit 50-person community of technicians, owners, collectors, enthusiasts
- Museums opening (AUTOPHOTO NYC, Photomatica SF) signal cultural importance
- Finding a working booth feels like discovering treasure

#### 5. **Cultural Significance**
- 2025 marks **100 years of the photobooth** (invented 1925 by Anatol Josepho)
- Woven into modern culture since the 1960s
- Iconic moments: The Beatles, John Lennon & Yoko Ono, the Kennedys
- Film appearances (cataloged by enthusiasts)
- International Photobooth Convention shows global community strength

### What Information Enthusiasts Want to Know

#### Primary Information (Must-Have)
1. **Machine Specifications**
   - Model name (e.g., "Auto Photo Model 11", "Photo-Me Model 9")
   - Manufacturer (Auto-Photo, Photo-Me, Photomatic)
   - Year manufactured
   - Serial number (if known)
   - Booth type (analog/chemical/digital/instant)

2. **Film & Output Details**
   - Photo type (B&W, color, or both)
   - Strip format (4-up, 2-up, wide format)
   - Chemistry used (dip-and-dunk, instant, etc.)
   - Development time
   - Image quality notes

3. **Practical Visit Information**
   - Operational status (active, temporarily closed, permanently closed)
   - Current cost
   - Hours of operation
   - Payment methods (cash only, card accepted)
   - Access instructions (inside venue, requires entry fee, etc.)

#### Secondary Information (High Value)
4. **Historical Context**
   - When this booth was installed
   - Previous locations (if moved)
   - Historical significance of the location
   - Notable visitors or moments
   - How long it's been operating

5. **Condition & Authenticity**
   - Original vs. restored parts
   - Maintenance status
   - Known issues or quirks
   - Who maintains it
   - Last service date

6. **Cultural Context**
   - Why this location is special
   - Neighborhood/venue history
   - Connection to local culture
   - Stories from visitors
   - Film/media appearances

#### Community Information (Engagement)
7. **User Contributions**
   - Photos of actual strips
   - Visit reports and reviews
   - Tips and tricks
   - Best time to visit
   - Nearby attractions

8. **Operator Story**
   - Who runs this booth
   - Their story/motivation
   - Other booths they operate
   - Years in business
   - Social media presence

### What Makes Someone WANT to Visit

#### Compelling Narratives
1. **The Machine's Story**
   - "This Model 11 has been in Times Square since 1967, photographing millions of visitors"
   - "One of only 3 working wide-format machines left in North America"
   - "Featured in [movie/TV show]"
   - "The last operating booth in [city/region]"

2. **The Location's Character**
   - "Hidden inside a vintage arcade with games from the 1980s"
   - "In a historic train station that's been serving travelers since 1903"
   - "Part of a family-owned diner that's a local institution"
   - "Tucked in a corner of a vibrant street market"

3. **The Experience Promise**
   - "True analog black & white strips ready in 3 minutes"
   - "Classic 4-shot vertical strip, just like your grandparents used"
   - "Color chemistry is rare - this is one of the few color booths left"
   - "The photos from this booth are good enough for passport photos"

#### Social Proof
4. **Community Validation**
   - User ratings and reviews
   - Recent visitor photos
   - Number of bookmarks/saves
   - "Popular" or "Trending" indicators
   - Featured in collections or guides

5. **Visual Appeal**
   - High-quality photos of the booth exterior
   - Interior shots showing the classic design
   - Sample strips showing output quality
   - Photos of happy visitors
   - Google photos showing the venue context

#### Practical Confidence
6. **Visitor Reassurance**
   - "Verified operational [date]"
   - Clear directions and access instructions
   - Known hours and cost
   - Tips from recent visitors
   - "What to expect" information

#### Discovery Features
7. **Related Content**
   - Nearby booths for "booth crawls"
   - Other booths with same model/manufacturer
   - Booths in the same city/neighborhood
   - Collections featuring this booth
   - Similar booths you might like

// Database function (Supabase migration)
CREATE OR REPLACE FUNCTION get_nearby_booths(
  p_latitude DECIMAL,
  p_longitude DECIMAL,
  p_radius_km INTEGER DEFAULT 5,
  p_limit INTEGER DEFAULT 6,
  p_exclude_booth_id UUID DEFAULT NULL
)
RETURNS TABLE (
  id UUID,
  name TEXT,
  slug TEXT,
  city TEXT,
  photo_exterior_url TEXT,
  ai_generated_image_url TEXT,
  latitude DECIMAL,
  longitude DECIMAL,
  distance_km DECIMAL
) AS $$
BEGIN
  RETURN QUERY
  SELECT
    b.id,
    b.name,
    b.slug,
    b.city,
    b.photo_exterior_url,
    b.ai_generated_image_url,
    b.latitude,
    b.longitude,
    ROUND(
      ST_Distance(
        b.location_geog,
        ST_SetSRID(ST_MakePoint(p_longitude, p_latitude), 4326)::geography
      ) / 1000,
      2
    ) AS distance_km
  FROM booths b
  WHERE
    b.id != COALESCE(p_exclude_booth_id, '00000000-0000-0000-0000-000000000000'::UUID)
    AND b.status = 'active'
    AND b.location_geog IS NOT NULL
    AND ST_DWithin(
      b.location_geog,
      ST_SetSRID(ST_MakePoint(p_longitude, p_latitude), 4326)::geography,
      p_radius_km * 1000
    )
  ORDER BY distance_km ASC
  LIMIT p_limit;
END;

// lib/recommendations/getSimilarBooths.ts

export async function getSimilarBooths(boothId: string, limit = 6) {
  const supabase = createClient();

  // Get source booth
  const { data: booth } = await supabase
    .from('booths')
    .select('*')
    .eq('id', boothId)
    .single();

  if (!booth) return [];

  // Scoring algorithm
  const { data: candidates } = await supabase
    .from('booths')
    .select('*')
    .neq('id', boothId)
    .eq('status', 'active');

  const scored = candidates?.map(candidate => {
    let score = 0;

    // Same machine model = +10 points
    if (candidate.machine_model === booth.machine_model) {
      score += 10;
    }

    // Same manufacturer = +5 points
    if (candidate.machine_manufacturer === booth.machine_manufacturer) {
      score += 5;
    }

    // Same photo type = +3 points
    if (candidate.photo_type === booth.photo_type) {
      score += 3;
    }

    // Same booth type = +3 points
    if (candidate.booth_type === booth.booth_type) {
      score += 3;
    }

    // Same country = +2 points
    if (candidate.country === booth.country) {
      score += 2;
    }

    // Similar cost = +1 point
    if (candidate.cost === booth.cost) {
      score += 1;
    }

    return { ...candidate, similarity_score: score };
  }) || [];

  // Sort by score and return top N
  return scored
    .sort((a, b) => b.similarity_score - a.similarity_score)
    .slice(0, limit);
}
